## 1. Overview

### 1.1 Purpose

영업직이 하루 동안 처리하는 **업무 체크리스트**, **상담 일정 관리**, **금일 종합보고**, **시간표 배정 자동화**를 하나의 Streamlit 앱에서 처리한다.

### 1.2 Background

- 현재 상담일정·업무 체크를 별도 툴(엑셀, 메모 등)로 관리
- 팀원 공유가 불편하고 일정 누락 발생
- 등록 후 시간표 배정 과정이 수동이라 오류 가능성 있음

---

## 2. Scope

### 2.1 In Scope

- [x] **업무 체크리스트**: 고정 항목 + 당일 추가 가능, 매일 자정 초기화
- [x] **상담일정 TO DO LIST**: 이름/일정/예정매출 입력·조회·삭제
- [x] **달력**: 상담 일정이 달력에 표시
- [x] **시계**: 현재 시간 실시간 표시
- [x] **금일 종합보고**: 오늘 상담 목록 + 예정매출 합계 + 전체 현황
- [x] **시간표 배정 자동화**: 이름/월/과목 입력 → 지정 양식으로 출력
- [x] **팀원 공유**: Streamlit Share URL 접속으로 실시간 공유
- [x] **트렌디한 UI**: 2024-2025 트렌드 반영 (미니멀, 카드 기반)

### 2.2 Out of Scope

- 로그인/권한 분리 (팀원 모두 동일 접근)
- 모바일 앱 (웹 반응형으로 대체)
- 알림/푸시 기능
- 매출 데이터 분석 차트 (추후 추가)

## 3. Requirements

### 3.1 Functional Requirements

| ID | 기능 | 우선순위 |
|----|------|----------|
| FR-01 | 고정 업무 항목 관리 (CRUD) + 매일 체크 초기화 | High |
| FR-02 | 당일 추가 업무 입력 + 체크 완료 처리 | High |
| FR-03 | 상담 일정 입력 (이름, 날짜/시간, 예정매출) | High |
| FR-04 | 상담 일정 달력 표시 | High |
| FR-05 | 금일 상담 목록 + 예정매출 합산 보고 | High |
| FR-06 | 전체 상담 현황 요약 (월별 예정매출 등) | Medium |
| FR-07 | 시간표 배정 자동화 (입력 → 양식 출력) | High |
| FR-08 | 배정 결과 복사/출력 기능 | Medium |
| FR-09 | 현재 시간 표시 (시계) | Low |
| FR-10 | 달력에 오늘 하이라이트 + 상담 일정 마커 | Medium |

### 3.2 Non-Functional Requirements

| Category | 기준 |
|----------|------|
| 성능 | 페이지 로드 3초 이내 |
| 공유 | Streamlit Share URL 1개로 팀 전체 접근 |
| DB | SQLite (Streamlit Cloud 재시작 시 초기화 — 허용 범위) |
| 배포 | GitHub → Streamlit Share 자동 배포 |

---

## 4. Success Criteria

- [ ] FR-01~FR-10 전부 동작
- [ ] 팀원이 URL 접속으로 상담 일정 실시간 확인 가능
- [ ] 시간표 배정 입력 → 양식 출력 정확도 100%
- [ ] 트렌디한 UI — 카드 기반, 깔끔한 컬러 팔레트

---

## 5. Risks and Mitigation

| Risk | 영향 | 대처 |
|------|------|------|
| Streamlit Cloud DB 초기화 | Medium | 매일 체크리스트는 초기화되어도 무방; 상담 데이터는 재입력 필요 → 허용 |
| 시간표 양식 미확정 | High | 양식 수령 후 FR-07 구현 — 양식 확정 전까지 UI 먼저 구현 |
| 팀원 동시 수정 충돌 | Low | SQLite WAL 모드 사용 |

## 6. UI/UX 방향

### 6.1 디자인 레퍼런스 (2025 트렌드)

- **컬러**: 화이트 베이스 + 네이비/딥그린 포인트 (Notion, Linear 스타일)
- **레이아웃**: 카드 그리드, 명확한 섹션 구분
- **타이포**: 한글 가독성 우선, 굵은 수치, 가벼운 레이블
- **상태 표시**: 컬러 배지 (완료=초록, 대기=회색, 오늘=파랑)

### 6.2 화면 구성 (4 섹션)

```
┌─────────────────────────────────────────────────────────┐
│  [시계]  [날짜]          금일 종합보고 요약 카드들        │
├───────────────────┬─────────────────────────────────────┤
│  업무 체크리스트   │  상담 일정 TO DO LIST               │
│  · 고정 항목      │  · 이름 / 날짜 / 예정매출            │
│  · 오늘 추가      │  · 달력 뷰                          │
├───────────────────┴─────────────────────────────────────┤
│  시간표 배정 자동화                                       │
│  이름 / 월 / 과목 입력 → 양식 자동 출력                  │
└─────────────────────────────────────────────────────────┘
```
